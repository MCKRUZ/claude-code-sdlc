# Close-Gate Evidence
<!-- Phase C — Close & Transfer | Required artifact -->

> THE close gate: the client team ran one **real** spec end-to-end — triage → spec → delegate →
> grade → merge → deploy — **without the pod driving**. Observed, unassisted. Any help voids the
> run; a wobbly run is re-run on a *different* real spec. A toy spec proves nothing.

## Shadow flip (precondition)
Client engineers completed ≥3 real specs as Orchestrators with pod Checkers:
| Spec | Client Orchestrator | Pod Checker | Coaching needed (each = a transfer gap) |
|------|---------------------|-------------|------------------------------------------|

- Merge that deployed: [commit / PR link]    Deployed to: [env]

- Timestamps: triage [..] → merge [..] → deploy [..]
- Outcome: [passed unassisted / re-run required → which spec]
